# Math & Markdown Demo

This post focuses on **mathematical notation** rendered directly in the reading view.

## Inline Examples

We can write things like $a^2 + b^2 = c^2$ or $\nabla f(x)$ inline with the surrounding text.

The soft default theme should keep everything readable and quiet enough for dense notes.

## Display Equations

A standard Gaussian integral:

$$
\int_{-\infty}^{+\infty} e^{-x^2} \, dx = \sqrt{\pi}
$$

A tiny optimization problem:

$$
\min_{w} \; \mathcal{L}(w) = \sum_{i=1}^{n} \ell\big(y_i, f(x_i; w)\big) + \lambda \|w\|_2^2
$$

## Lists and Emphasis

- Use lists to separate ideas.
- Combine **bold** and *italic* with formulas, e.g. $\alpha$, $\beta$, $\gamma$.
- Let the layout stay simple so that the math remains the focus.
